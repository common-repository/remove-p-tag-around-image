== Description ==

This Plugin Removes paragraph tag from around your images which Wordpress puts by default.

== Installation ==

1. Copy the "remove-p-tag-around-image" folder into wp-content/plugins
2. Activate the plugin through the Plugins menu
3. That's It, No configration Required
